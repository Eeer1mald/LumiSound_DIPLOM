# Макеты экранов приложения LumiSound

Одним из ключевых этапов в разработке приложения является создание макета. На начальных этапах разработки макеты применяются для формирования прототипа приложения и визуальной демонстрации работы будущего продукта. Прототип представляет собой упрощённое изображение приложения, состоящее из последовательности нарисованных экранов. Внешний вид прототипа аналогичен тому, каким будет разрабатываемое приложение. Прототип должен включать основные элементы управления, эффективно моделировать переходы между экранами, обеспечивая наглядную демонстрацию внешнего вида и функционала будущего приложения.

Дизайн интерфейса мобильного приложения выполнен в стиле Material Design. Material Design — дизайн программного обеспечения и приложений операционной системы Android от компании Google. [4]

Модели экранов были сделаны при помощи веб-инструмента Figma. [5]

## Главный экран приложения

После успешной авторизации пользователь попадает на главный экран приложения. Главный экран содержит:
- заголовок «LumiSound»;
- кнопку случайного воспроизведения трека;
- кнопку создания плейлиста или синтеза;
- блок статистики активности пользователя за неделю (рецензии, оценки, комментарии);
- секцию плейлистов с сеткой превью;
- горизонтальный список «Недавно слушал»;
- нижнюю навигационную панель с вкладками «Главная», «Поиск», «Оценки», «Профиль»;
- мини-плеер над навигационной панелью (при активном воспроизведении).

## Экран плеера

При нажатии на трек открывается полноэкранный плеер. Экран плеера содержит:
- обложку трека в высоком качестве;
- название трека и имя исполнителя;
- waveform-прогресс-бар с возможностью перемотки;
- кнопки управления воспроизведением: предыдущий трек, Play/Pause, следующий трек;
- плавающие комментарии других пользователей;
- кнопку добавления трека в плейлист;
- кнопку перехода к рецензиям;
- кнопку перехода к написанию рецензии.

Переключение между треками осуществляется горизонтальным свайпом, закрытие плеера — свайпом вниз.

## Экран настроек

Экран настроек содержит:
- секцию «Аккаунт»: отображение email, смена пароля;
- секцию «Внешний вид»: выбор темы оформления (тёмная, светлая, системная), смена иконки приложения;
- секцию «Воспроизведение»: автовоспроизведение, нормализация громкости, скорость воспроизведения;
- секцию «Звук»: эквалайзер, виртуализатор;
- таймер сна;
- секцию «Конфиденциальность»: публичность профиля;
- секцию «Данные»: очистка кэша;
