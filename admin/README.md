# LumiSound Admin Panel

Веб-панель администратора для управления данными LumiSound.

## Запуск

Просто открой `admin/index.html` в браузере — никакой сборки не нужно.

Или запусти локальный сервер:
```bash
# Python
python -m http.server 8080 --directory admin

# Node.js
npx serve admin
```

Затем открой http://localhost:8080

## Возможности

- **Дашборд** — статистика по всем таблицам
- **Пользователи** — просмотр, редактирование, удаление профилей
- **Артисты** — создание/редактирование собственных артистов с загрузкой фото
- **Треки** — создание/редактирование треков с загрузкой аудио и обложки
- **Оценки** — просмотр и удаление оценок треков
- **Комментарии** — просмотр и удаление комментариев
- **Плейлисты** — просмотр и удаление плейлистов

## Авторизация

Войди через email/пароль своего Supabase аккаунта.

## Новые таблицы в БД

- `custom_artists` — собственные артисты (не из Audius)
- `custom_tracks` — треки собственных артистов
- Storage bucket `admin-uploads` — для загрузки файлов
