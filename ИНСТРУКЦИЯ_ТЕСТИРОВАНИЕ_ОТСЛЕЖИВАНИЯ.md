### 1. Обновите функции в Supabase

Выполните **только** части ШАГ 2 и ШАГ 3 из обновленного скрипта `update_tables_for_play_tracking.sql`:

```sql
-- ШАГ 2: Обновление функции для треков
CREATE OR REPLACE FUNCTION increment_track_play_count(
    p_user_id uuid,
    p_track_id text,
    p_track_title text,
    p_track_artist text,
    p_track_cover_url text DEFAULT NULL,
    p_track_preview_url text DEFAULT NULL
)
RETURNS void
LANGUAGE plpgsql
SECURITY DEFINER
AS $$
BEGIN
    INSERT INTO public.favorite_tracks (
        user_id, track_id, track_title, track_artist,
        track_cover_url, track_preview_url, play_count
    )
    VALUES (
        p_user_id, p_track_id, p_track_title, p_track_artist,
        NULLIF(p_track_cover_url, ''), NULLIF(p_track_preview_url, ''), 1
    )
    ON CONFLICT (user_id, track_id)
    DO UPDATE SET 
        play_count = favorite_tracks.play_count + 1,
        track_title = EXCLUDED.track_title,
        track_artist = EXCLUDED.track_artist,
        track_cover_url = COALESCE(NULLIF(EXCLUDED.track_cover_url, ''), favorite_tracks.track_cover_url),
        track_preview_url = COALESCE(NULLIF(EXCLUDED.track_preview_url, ''), favorite_tracks.track_preview_url);
END;
$$;

-- ШАГ 3: Обновление функции для артистов
CREATE OR REPLACE FUNCTION increment_artist_play_count(
    p_user_id uuid,
    p_artist_id text,
    p_artist_name text,
    p_artist_image_url text DEFAULT NULL
)
RETURNS void
LANGUAGE plpgsql
SECURITY DEFINER
AS $$
BEGIN
    INSERT INTO public.favorite_artists (
        user_id, artist_id, artist_name, artist_image_url, play_count
    )
    VALUES (
        p_user_id, p_artist_id, p_artist_name,
        NULLIF(p_artist_image_url, ''), 1
    )
    ON CONFLICT (user_id, artist_id)
    DO UPDATE SET 
        play_count = favorite_artists.play_count + 1,
        artist_name = EXCLUDED.artist_name,
        artist_image_url = COALESCE(NULLIF(EXCLUDED.artist_image_url, ''), favorite_artists.artist_image_url);
END;
$$;
```

## Устранение проблем

### Трек не появляется в профиле

1. **Проверьте логи** - есть ли ошибки при вызове RPC?
2. **Проверьте базу данных** - появилась ли запись в `favorite_tracks`?
3. **Проверьте авторизацию** - убедитесь, что вы залогинены
4. **Проверьте время прослушивания** - минимум 30 секунд или 50% трека

### Ошибка при вызове RPC

1. **Проверьте имя функции** - должно быть `increment_track_play_count`
2. **Проверьте параметры** - все параметры должны быть переданы
3. **Проверьте права доступа** - функция должна быть `SECURITY DEFINER`

### Функции не обновляются

1. Убедитесь, что вы выполнили `CREATE OR REPLACE FUNCTION`
2. Проверьте, что нет синтаксических ошибок
3. Попробуйте выполнить функцию вручную через SQL Editor:

```sql

    NULL,
    NULL

```
