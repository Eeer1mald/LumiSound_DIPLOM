# 🚀 Быстрый старт LumiSound

## Что это?

**LumiSound** - премиальное мобильное музыкальное приложение с уникальной системой оценивания треков (1-10).

## ⚡ Запуск за 30 секунд

Приложение готово к использованию! Просто откройте `/App.tsx` и увидите интерактивную версию.

### 📱 Что вы увидите:
- Главный экран с рекомендациями
- Полная навигация между экранами
- Систему оценок треков
- Статистику и аналитику

## 🎯 Основные возможности

### 1️⃣ Навигация
Используйте нижнюю панель для переключения между:
- **Домой** - главный экран с рекомендациями
- **Поиск** - поиск треков и жанров
- **Плейлисты** - ваши коллекции музыки
- **Профиль** - статистика и настройки

### 2️⃣ Воспроизведение
- Кликните на любой трек → откроется экран воспроизведения
- Оцените трек по шкале 1-10
- Используйте элементы управления

### 3️⃣ Оценки
- Профиль → "Мои оценки"
- Просмотр всех оценённых треков
- Фильтрация по рейтингу
- Статистика вкуса

## 🎯 Экраны приложения

### 🏠 Главный экран
- Приветствие пользователя
- Поле поиска
- Рекомендации (6 треков)
- Плейсхолдеры для будущих функций

### 🎵 Воспроизведение
- Крупная обложка
- Прогресс-бар
- Управление (play/pause/next/prev)
- ⭐ **Система оценок 1-10**
- Лайк и добавление в плейлист

### ⭐ Оценки
- Список оценённых треков
- Статистика (всего / средняя / топ)
- Фильтры по рейтингу
- Клик на трек → воспроизведение

### 🔍 Поиск
- Поле поиска с автофокусом
- Фильтры (треки/артисты/альбомы)
- Топ-5 трендовых треков
- 6 жанров

### 📂 Плейлисты
- 3 готовых плейлиста
- Кнопка создания нового
- Empty state для пустого списка

### 👤 Профиль
- Аватар пользователя
- 4 статистики (треки/оценки/плейлисты/лайки)
- Переходы (оценки/избранное/история)
- Достижения (8 badges)
