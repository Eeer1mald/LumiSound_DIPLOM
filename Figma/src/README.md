# LumiSound - Музыкальное потоковое приложение

Премиальный музыкальный стриминговый сервис с системой оценивания треков для платформы Android.

## 🚀 Быстрый старт

Приложение готово к использованию! По умолчанию показывается интерактивная версия с полной навигацией.

### Режимы просмотра

**Текущий режим**: Интерактивное приложение с навигацией между экранами.

Чтобы увидеть все экраны одновременно, замените содержимое `/App.tsx`:

```tsx
import { AllScreensDemo } from './components/AllScreensDemo';

export default function App() {
  return <AllScreensDemo />;
}
```

Чтобы увидеть экраны авторизации (Welcome, Login, SignUp):

```tsx
import { AuthScreens } from './components/auth/AuthScreens';

export default function App() {
  return <AuthScreens />;
}
```

## 📱 Экраны

### 1. Главный экран (Home)
- Персональное приветствие
- Поиск музыки
- Рекомендации треков
- Недавно прослушанное

### 2. Экран воспроизведения (Now Playing)
- Крупная обложка трека
- Элементы управления (Play/Pause, Next/Previous)
- Прогресс-бар
- **Система оценки 1-10** - ключевая функция приложения
- Кнопки Like и добавления в плейлист

### 3. Экран оценок (Ratings)
- Список всех оценённых треков
- Статистика (всего оценок, средняя оценка, топ треков)
- Фильтрация по оценкам
- Аналитика музыкального вкуса

### 4. Поиск (Search)
- Поле поиска с автофокусом
- Фильтры (треки, артисты, альбомы)
- Трендовые треки
- Жанры с градиентными карточками

### 5. Плейлисты (Playlists)
- Список пользовательских плейлистов
- Создание новых плейлистов
- Карточки с обложками

### 6. Профиль (Profile)
- Аватар пользователя
- Статистика прослушивания
- Переходы к оценкам, любимым трекам, истории
- Достижения

## 🔧 Технологии

- React
- TypeScript

- Lucide React (иконки)

## 📦 Компоненты

### Основные компоненты
- `HomeScreen` - главный экран
- `NowPlayingScreen` - экран воспроизведения с системой оценок
- `RatingsScreen` - экран оценок
- `SearchScreen` - поиск
- `PlaylistsScreen` - плейлисты
- `ProfileScreen` - профиль

### UI компоненты
- `MobileFrame` - рамка мобильного устройства
- `BottomNavigation` - нижняя навигация
- `SearchBar` - поле поиска
- `TrackCard` - карточка трека
- `LumiSoundLogo` - логотип

### Авторизация (в папке /components/auth)
- `WelcomeScreen` - экран приветствия
- `LoginScreen` - вход
- `SignUpScreen` - регистрация

## 🎯 Ключевые особенности

1. **Система оценок 1-10** - уникальная функция, позволяющая пользователям детально оценивать треки
2. **Премиальный дизайн** - конкурирует с Spotify и Apple Music
3. **Персонализация** - рекомендации на основе оценок
4. **Атмосферный UI** - темная тема с градиентами и glow-эффектами
5. **Плавная навигация** - интуитивный UX
