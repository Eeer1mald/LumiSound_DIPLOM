# Инструкция по использованию

## Режимы просмотра

### 1. Основное приложение (текущий режим)
**Файл**: `/App.tsx`

Полноценное интерактивное приложение с навигацией между экранами:
- ✅ Навигация между экранами
- ✅ Воспроизведение треков
- ✅ Система оценок
- ✅ Полный функционал

**Как использовать:**
- Используйте нижнюю навигацию для перехода между экранами
- Кликайте на треки для воспроизведения
- Оценивайте треки по шкале 1-10
- Просматривайте статистику в профиле

### 2. Демонстрация всех экранов
**Файл**: `/components/AllScreensDemo.tsx`

Показывает все экраны одновременно для дизайн-ревью:

```tsx
import { AllScreensDemo } from './components/AllScreensDemo';

export default function App() {
  return <AllScreensDemo />;
}
```

### 3. Экраны авторизации
**Файл**: `/components/auth/AuthScreens.tsx`

Показывает экраны Welcome, Login, Sign Up:

```tsx
import { AuthScreens } from './components/auth/AuthScreens';

export default function App() {
  return <AuthScreens />;
}
```

## Навигация в приложении

### Главный экран (Home)
- Просмотр рекомендаций
- Поиск музыки
- Клик на трек → переход к воспроизведению

### Поиск (Search)
- Просмотр трендов
- Фильтрация по категориям
- Обзор жанров

### Плейлисты (Playlists)
- Просмотр плейлистов
- Создание нового плейлиста

### Профиль (Profile)
- Статистика прослушивания
- Переход к оценкам
- Настройки

### Воспроизведение (Now Playing)
- Управление воспроизведением
- **Оценка трека 1-10**
- Добавление в избранное
- Кнопка назад (chevron down)

### Оценки (Ratings)
- Список оценённых треков
- Фильтрация по оценкам
- Статистика (средняя оценка, топ треки)

## Моковые данные

Приложение использует моковые данные:
- 6 треков с обложками из Unsplash
- 4 оценённых трека
- 3 плейлиста
- Статистика пользователя

## Следующие шаги для интеграции

1. **База данных**: Подключить Supabase или другую БД
2. **API**: Создать эндпоинты для треков, плейлистов, оценок
3. **Аудио**: Интегрировать Web Audio API или Howler.js
4. **Поиск**: Реализовать поиск по базе данных
5. **Авторизация**: Подключить экраны Welcome/Login/SignUp к реальной аутентификации
